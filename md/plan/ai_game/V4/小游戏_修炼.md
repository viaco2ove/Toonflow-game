# 修炼小游戏设计:[小游戏设计.md](../V3/%E5%B0%8F%E6%B8%B8%E6%88%8F%E8%AE%BE%E8%AE%A1/%E5%B0%8F%E6%B8%B8%E6%88%8F%E8%AE%BE%E8%AE%A1.md)

第一进入这个小游戏后，那个小游戏面板应该是折叠起来的，
然后旁白应该问我修炼什么，是否需要陪练。 需要陪练就可以选择一个角色进行指导。可以修炼功法，技能。 如果物品技能什么都没有。那就是基础功法和基础体术修炼和基础冥想。  修炼后给予经验（随机少于等级*50），功法等级升级，技能升级等奖励。
例如运行炼炎决后，经验+1，炼炎决（lv1）没有变化。然后要立即把用户的角色参数卡信息。
