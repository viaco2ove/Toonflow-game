# 小游戏 挖矿
规则:
- 第一进入这个小游戏后，那个小游戏面板应该是折叠起来的，
然后旁白应该问我准备挖什么，是否需要陪练。 需要陪练就可以选择一个角色进行陪练。
- 挖矿后，获得目标矿物，偶尔会挖出各种宝物

# 小游戏 升级装备
规则:
- 第一进入这个小游戏后，那个小游戏面板应该是折叠起来的，
然后旁白应该问我准备升级什么装备，是否需要找人升级装备。 需要找人升级装备就可以选择一个角色来帮忙升级装备。
- 用户只能升级等级小于或等于自身等级的装备。
用户默认拥有装备强化术（lv1）。
用户可以使用物品栏里的矿石、强化石、1金币升级装备。
技能也能升级，最高升到角色当前等级，升级技能需要20金币
装备强化术：可以提升物品栏里的装备等级，例如铁剑（lv1）可以升级为铁剑（lv2），升级后的装备等级必须小于或等于用户等级，且小于或等于装备强化术等级。
每次升级装备扣取金钱1，并扣取对应材料。
例如:铁剑（lv1）升级成功，铁剑（lv2）+1，铁剑（lv1）-1。

# 小游戏 研发技能
规则:
- 第一进入这个小游戏后，那个小游戏面板应该是折叠起来的，
然后旁白应该问我准备研发什么技能，是否需要陪练。 需要陪练就可以选择一个角色进行陪练。
- 每次研发技能都会消耗5金币
- 可以研发任意技能，但是于用户已有技能和物品关系越少升功率越低。
例如用户有灭魔尺和炼炎决，那么炼炎灭魔尺技能成功率很高。
例如用户想研发吞噬技能，但是物品和已有技能都没有相关的就成功率很低。
