# 本地头像分离模型安装

本文档记录当前项目里两个本地头像分离模型的安装方式：

- `BiRefNet 本地`
- `MODNet 本地`

项目当前实现目录：

- 根目录：`Toonflow-game/tools/avatar-matting/birefnet`
- Python 虚拟环境：`Toonflow-game/tools/avatar-matting/birefnet/venv`
- 模型缓存目录：`Toonflow-game/tools/avatar-matting/birefnet/model-cache`

## 通用前置

先准备 Python 3。

Windows PowerShell：

```powershell
python --version
```

如果 `python` 不可用，也可以用：

```powershell
py -3 --version
```

创建虚拟环境：

```powershell
python -m venv Toonflow-game/tools/avatar-matting/birefnet/venv
```

激活虚拟环境：

```powershell
Toonflow-game/tools/avatar-matting/birefnet/venv/Scripts/Activate.ps1
```

升级基础打包工具：

```powershell
python -m pip install --upgrade pip setuptools wheel
```

安装项目当前使用的本地头像分离公共依赖：

```powershell
python -m pip install rembg==2.0.67 onnxruntime==1.22.1 pillow numpy
```

## BiRefNet 本地

### 依赖安装命令

`BiRefNet` 目前直接复用 `rembg` 的模型体系，所以不需要手动单独下载 ONNX 文件。

安装命令就是：

```powershell
python -m pip install rembg==2.0.67 onnxruntime==1.22.1 pillow numpy
```

### 预热命令

项目运行时会生成 `run_birefnet.py`。如果你要手动预热模型，可以执行：

```powershell
python Toonflow-game/tools/avatar-matting/birefnet/run_birefnet.py --warmup --model birefnet-portrait
```

可选模型：

- `birefnet-portrait`
- `birefnet-general`
- `birefnet-general-lite`

例如：

```powershell
python Toonflow-game/tools/avatar-matting/birefnet/run_birefnet.py --warmup --model birefnet-general
```

## MODNet 本地

### 依赖安装命令

`MODNet` 使用 `onnxruntime + pillow + numpy`，依赖命令如下：

```powershell
python -m pip install onnxruntime==1.22.1 pillow numpy
```

如果你也要保留和项目一致的完整公共依赖，可以直接继续使用：

```powershell
python -m pip install rembg==2.0.67 onnxruntime==1.22.1 pillow numpy
```

### 模型下载命令

项目当前优先使用 Hugging Face 直链下载：

```powershell
python -c "import pathlib, urllib.request; p=pathlib.Path(r'Toonflow-game/tools/avatar-matting/birefnet/model-cache/modnet_photographic_portrait_matting.onnx'); p.parent.mkdir(parents=True, exist_ok=True); urllib.request.urlretrieve('https://huggingface.co/DavG25/modnet-pretrained-models/resolve/main/models/modnet_photographic_portrait_matting.onnx', p)"
```

下载后的模型文件应位于：

```text
Toonflow-game/tools/avatar-matting/birefnet/model-cache/modnet_photographic_portrait_matting.onnx
```

### 预热命令

项目运行时会生成 `run_modnet.py`。如果你要手动预热模型，可以执行：

```powershell
python Toonflow-game/tools/avatar-matting/birefnet/run_modnet.py --warmup --model-path Toonflow-game/tools/avatar-matting/birefnet/model-cache/modnet_photographic_portrait_matting.onnx
```

## 一次性完整安装示例

### BiRefNet（质量高,资源消耗大，对于常规机器速度非常慢，性价比略低）

```powershell
python -m venv Toonflow-game/tools/avatar-matting/birefnet/venv
Toonflow-game/tools/avatar-matting/birefnet/venv/Scripts/Activate.ps1
python -m pip install --upgrade pip setuptools wheel
python -m pip install rembg==2.0.67 onnxruntime==1.22.1 pillow numpy
python Toonflow-game/tools/avatar-matting/birefnet/run_birefnet.py --warmup --model birefnet-portrait
```

### MODNet（质量一般偏高,资源消耗小，速度快，性价比更高）

```powershell
python -m venv Toonflow-game/tools/avatar-matting/birefnet/venv
Toonflow-game/tools/avatar-matting/birefnet/venv/Scripts/Activate.ps1
python -m pip install --upgrade pip setuptools wheel
python -m pip install onnxruntime==1.22.1 pillow numpy rembg==2.0.67
python -c "import pathlib, urllib.request; p=pathlib.Path(r'Toonflow-game/tools/avatar-matting/birefnet/model-cache/modnet_photographic_portrait_matting.onnx'); p.parent.mkdir(parents=True, exist_ok=True); urllib.request.urlretrieve('https://huggingface.co/DavG25/modnet-pretrained-models/resolve/main/models/modnet_photographic_portrait_matting.onnx', p)"
python Toonflow-game/tools/avatar-matting/birefnet/run_modnet.py --warmup --model-path Toonflow-game/tools/avatar-matting/birefnet/model-cache/modnet_photographic_portrait_matting.onnx
```
